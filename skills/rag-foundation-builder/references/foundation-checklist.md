# Foundation Checklist

1. Define user and task boundary.
2. Define non-goals to avoid scope creep.
3. Create 20-50 representative questions.
4. Annotate expected evidence sources for each question.
5. Implement ingest and document normalization.
6. Select chunking baseline and document exceptions.
7. Build vector index with metadata fields.
8. Implement retrieval endpoint with deterministic params.
9. Build generation prompt with citation requirement.
10. Run baseline and capture failure examples.
11. Write first iteration log entry.

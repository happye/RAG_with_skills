# Observability

## Required Telemetry

- Query classification result.
- Retrieved chunk IDs and scores.
- Prompt token budget and final context size.
- Answer with citations and confidence tag.
- End-to-end latency and cost.

## Alerts

- Retrieval recall trend drop.
- Faithfulness drop on canary set.
- Latency p95 breach.
- Cost per successful answer drift.
